ADR-014

Multi-Cloud Orchestration Layer

Accepted

Decision

Adopt HashiCorp Nomad for cross-region scheduling to reduce dependency on vendor-locked Kubernetes distributions while maintaining hybrid-cloud portability.

Alternatives

  • EKS/GKE Hybrid
  • Bare Metal K8s

Consequences

Increased training cost for infrastructure team.

schedule Oct 24, 2023

Event-Driven Security Auditing

Proposed

Decision

Implement a unified NATS-based event stream for all infrastructure mutations to enable real-time compliance monitoring and automated rollback.

  • Webhook polling
  • CloudTrail only

RFC-204

Zero-Trust Identity Mesh Integration

Under Review
+4
report_problem Problem Statement

Current internal services rely on perimeter security (VPN). This is insufficient for the upcoming Phoenix-Edge node deployment where nodes exist on untrusted external networks.

rocket Proposed Solution

Implementing SPIFFE/Spire as the identity provider for all service-to-service communication, coupled with a WireGuard-based overlay mesh for encrypted transit.

MVP Scope
  • Core Spire server in US-East-1
  • Sidecar injection for 3 pilot services
  • Certificate rotation within 1hr TTL
Open Questions

"How do we handle legacy Windows Server 2019 nodes that don't support the SPIFFE Unix Domain Socket natively?"
