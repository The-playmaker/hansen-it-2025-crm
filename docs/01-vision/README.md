# 01 Vision

## Project Phoenix

Project Phoenix er en langsiktig plattform for Hansen IT. Målet er å samle drift, sikkerhet, dokumentasjon, kundedialog, rapportering og AI i én helhetlig arbeidsflate.

## Visjon

> Hansen IT skal bli kjent som selskapet som finner, forstår og løser IT-problemene andre ikke ser.

Phoenix skal gjøre dette mulig ved å kombinere:

- praktisk IT-feilsøking
- sikkerhetsanalyse
- dokumentasjon
- automatisering
- AI-støtte
- kundeportal og rapporter

## Misjon

Hansen IT skal hjelpe små og mellomstore bedrifter med stabil, sikker og forståelig IT-drift.

Phoenix skal støtte hele kundereisen:

```text
Første henvendelse
→ lead
→ kunde
→ analyse
→ tilbud
→ leveranse
→ dokumentasjon
→ overvåking
→ kontinuerlig forbedring
```

## Hva Phoenix ikke er

Phoenix skal ikke bli en tilfeldig samling funksjoner. Hver modul skal løse et tydelig problem og passe inn i helheten.

Phoenix skal heller ikke prøve å erstatte alle spesialverktøy. Der det finnes gode løsninger, skal Phoenix integrere dem. Der Hansen IT har unik arbeidsflyt eller verdi, kan Phoenix bygge egne moduler.

## Kjerneidé

> Finn problemet. Forstå årsaken. Løs det riktig. Dokumenter det. Automatiser det. Forbedre det.
